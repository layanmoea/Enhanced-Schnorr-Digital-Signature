# Enhanced-Schnorr-Digital-Signature

## Overview
This project is a university implementation of an enhanced Schnorr digital signature scheme using Java and the Bouncy Castle cryptography library.

## Features
- Elliptic Curve Cryptography (secp256k1)
- SHA3-256 hashing
- Deterministic nonce generation
- Public key binding
- Compressed point encoding
- Signature generation
- Signature verification

## Technologies
- Java
- Bouncy Castle
- Elliptic Curve Cryptography (ECC)
- SHA3-256

## Requirements
- Java
- Bouncy Castle library

## Note
This project was developed for educational purposes as part of a university cryptography project.
